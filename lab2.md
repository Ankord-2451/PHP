1. Какие способы установки PHP существуют?

Существует несколько основных способов установки PHP:

1) Установка через официальный сайт

Скачать установщик с официального сайта PHP
Подходит для Windows. Устанавливается как обычная программа.

2) Установка через пакетный менеджер (Linux / macOS)

Ubuntu / Debian:

sudo apt install php


Fedora:

sudo dnf install php


Arch:

sudo pacman -S php


macOS (через Homebrew):

brew install php


Это самый удобный способ для Linux.

3) Установка в составе веб-сервера (готовые сборки)

XAMPP

OpenServer

WampServer

Подходит для обучения и разработки. Устанавливается сразу Apache + MySQL + PHP.

4) Установка через Docker

Используется в современной разработке:

docker pull php


Позволяет запускать PHP в контейнере без установки в систему.

2. Как проверить, что PHP установлен и работает?
Способ 1 — через консоль

Ввести команду:

php -v


Если PHP установлен — появится версия, например:

PHP 8.3.2 (cli)

Способ 2 — через браузер

Создать файл test.php

Вписать в него:

<?php
phpinfo();
?>


Открыть файл через сервер (например, http://localhost/test.php
)

Если PHP работает — появится страница с информацией о конфигурации.

3. Чем отличается echo от print?
Характеристика	echo	print
Тип	языковая конструкция	языковая конструкция
Возвращает значение	нет	да (возвращает 1)
Можно выводить несколько строк	да	нет
Скорость	немного быстрее	немного медленнее
Пример:
echo "Hello";
print "Hello";

Главное отличие:

echo ничего не возвращает

print возвращает 1, поэтому его можно использовать в выражениях

Пример:

if (print "Hello") {
    echo " World";
}
