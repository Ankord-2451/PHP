Контрольные вопросы

В чем разница между for, while и do-while?

for Используется, когда заранее известно количество повторений.
Удобен, если есть счетчик.

Вся логика (инициализация, условие, шаг) находится в одной строке.
Пример: перебор массива, фиксированное количество итераций.

while Используется, когда количество повторений заранее неизвестно.
Цикл выполняется, пока условие истинно.
Сначала проверяется условие, потом выполняется код.
Пример: чтение данных, пока не достигнут конец файла.

do-while Сначала выполняется код, потом проверяется условие.
Гарантированно выполняется минимум один раз.
Пример: меню программы, которое должно показаться хотя бы один раз.

2️ Как работает тернарный оператор ? : в PHP?

Это короткая форма if-else.

Синтаксис:

условие ? значение_если_true : значение_если_false;

Если условие истинно — возвращается первое значение, иначе второе.

3️ Что произойдет, если в do-while условие изначально ложно?

Код внутри do выполнится один раз,
потому что проверка условия происходит после выполнения цикла.

Даже при ложном условии тело цикла выполнится один раз.
