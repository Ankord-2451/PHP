# 🚀 Запуск и использование Recipe Manager (Modernized)

## Системные требования

- PHP 8.0+ (для типизации)
- Веб-сервер или встроенный PHP-сервер

## Быстрый старт

### 1. Перейти в директорию проекта
```bash
cd LAB7
```

### 2. Запустить встроенный веб-сервер
```bash
php -S localhost:8000
```

Сервер запустится на `http://localhost:8000`

### 3. Открыть в браузере

**Нативный PHP режим:**
```
http://localhost:8000/index.php
```

**Twig режим:**
```
http://localhost:8000/index.php?mode=twig
```

## 📝 Описание режимов

### Нативный PHP режим
**URL:** `http://localhost:8000/index.php`

**Характеристики:**
- Использует шаблоны из папки `templates/`
- Полный контроль над HTML
- Защита от XSS через `htmlspecialchars()`
- Простота и скорость
- Идеален для малых проектов

**Файлы:**
- `templates/layout.php` - HTML обертка
- `templates/list.php` - Таблица рецептов
- `templates/form.php` - Форма добавления/редактирования

### Twig режим
**URL:** `http://localhost:8000/index.php?mode=twig`

**Характеристики:**
- Использует шаблоны из папки `templates_twig/`
- Мощный шаблонизатор Twig
- Наследование шаблонов
- Поддержка фильтров (например, `format_duration`)
- Лучшее разделение логики и представления
- Идеален для сложных интерфейсов

**Файлы:**
- `templates_twig/base.html.twig` - базовый макет
- `templates_twig/list.html.twig` - список с наследованием
- `templates_twig/form.html.twig` - форма рецепта

## 📊 Кастомный фильтр Twig

**Фильтр:** `format_duration`

**Назначение:** Преобразует минуты в удобный формат

**Примеры:**
```
45 мин  → "45 мин"
60 мин  → "1 ч"
90 мин  → "1 ч 30 мин"
150 мин → "2 ч 30 мин"
180 мин → "3 ч"
```

**Использование в Twig:**
```twig
{{ recipe.prepTime | format_duration }}
```

## 🔧 Методы Recipe

Класс `Recipe` содержит следующие методы-геттеры:

```php
$recipe->getName()         // Возвращает название
$recipe->getPrepTime()     // Возвращает время в минутах (int)
$recipe->getIngredients()  // Возвращает ингредиенты
$recipe->getInstructions() // Возвращает инструкции
$recipe->getCategory()     // Возвращает категорию
$recipe->getDifficulty()   // Возвращает сложность
$recipe->getCreatedAt()    // Возвращает дату создания
$recipe->getAuthor()       // Возвращает автора
```

## 🐛 Решение проблем

### Ошибка: "Template not found"
- Убедитесь, что сервер запущен в корне проекта `LAB7/`
- Проверьте, что папки `templates/` и `templates_twig/` существуют

### Ошибка: "Class not found"
- Убедитесь, что файл `vendor/autoload.php` существует
- Проверьте, что все классы в `vendor/twig/twig/src/` на месте

### Рецепты не загружаются
- Проверьте, что `data.json` существует и содержит валидный JSON
- Используйте `test-modes.php` для проверки
